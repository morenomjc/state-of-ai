## Viewing

Open any `.md` file in [VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) with the Marp extension, or use the CLI:

```bash
npx @marp-team/marp-cli ai-state-slides.md --preview
```

## Exporting

```bash
# PDF
npx @marp-team/marp-cli ai-state-slides.md --pdf

npx @marp-team/marp-cli ai-state-slides.md --html
```
